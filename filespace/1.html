支持的格式：jpg，jpeg，png，webp，avif，ico，bmp，svg，gif，tga，raw，psd，apng，tif，tiff，mp3，wav，ogg，flac，mid，mpeg，wma，aac，mp4，wmv，avi，3gp，3gpp，mkv，mov，flv，swf，rmvb，webm，ppt，pptx，doc，docx，xls，xlsx，pdf，m3u8，m3u，ts，txt，js，json，css，ttf，otf，woff，woff2，zip，rar，7z，apk，exe，msi，ipa，gz，htm，mcworld，mcaddon，mcpack

使用方法

  • 点击上方的 1.文件上传 上传文件，上传成功后会返回一个json，记下 id 的值，之后打开 2.开启直链 把刚才记下的 id 的值填入页面的框里，之后点提交，会返回一个json，其中 downloadUrl 的值即为文件的直链，直链需等待1-2分钟后才可访问
  • 建议保存第一步时记下的 id 的值方便后续管理文件
